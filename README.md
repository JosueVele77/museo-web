El sitio web del **Museo Nacional del Ecuador** es una aplicación frontend diseñada para proporcionar información detallada sobre el museo, su historia y sus exposiciones de manera accesible, atractiva y responsiva.

## 📂 Estructura del Proyecto

El proyecto está organizado de la siguiente manera para un fácil mantenimiento y escalabilidad:

```text
museo-web/
├── docs/
│   ├── css/
│   │   ├── language-selector.css  # Estilos para el selector de idioma
│   │   └── style.css              # Estilos principales del sitio
│   ├── img/                       # Imágenes del sitio y banners
│   │   ├── historia1.webp
│   │   ├── ...
│   ├── js/
│   │   ├── scripts.js             # Lógica principal, interactividad y evaluación
│   │   └── translations.js        # Diccionario para el soporte multi-idioma
│   ├── pages/
│   │   ├── evaluacion.html        # Página del cuestionario interactivo
│   │   ├── historia.html          # Página sobre la historia del museo
│   │   └── ubicacion.html         # Página con mapas y ubicación
│   └── index.html                 # Página de inicio
└── README.md
